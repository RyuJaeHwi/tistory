HTML 01. HTML 기초 문법
=
HTML 문법 - 태그
------------

> **태그란?**  
> HTML 코드에서 정보(콘텐츠)를 정의하는 형식

태그는 <> ~ </> 기호를 사용하여 콘텐츠 하나의 시작과 끝을 표시합니다.

태그에는 다양한 종류가 있으며, 태그명은 해당 콘텐츠의 성격과 어떤 콘텐츠인지 의미를 나타내게 됩니다.

브라우저에 어떻게 표시해야 하는지 알려주는 역할이라고 이해하면 쉽습니다!

***기본 태그 사용 방식***`<태그명> 콘텐츠 넣는 위치 </태그명>`  

태그는 보통 위 사용 방식 처럼 <> ~ </> 기호를 사용하여 시작과 끝을 표시하지만, 가끔 이렇게 시작과 끝을 구분하지 않아도 되는 태그도 존재합니다.

이는 **단일 태그**라고 합니다.

***단일 태그 사용 방식***`<태그명 />
또는
<태그명>`

단일 태그는 콘텐츠를 감싸는 것이 아니기 때문에, 시작과 끝을 구분할 필요도 없습니다!

HTML 문법 - 속성
------------

> **속성이란?**  
> 태그에 부가적인 기능을 정의하는 것 (선택 사항)

속성은 시작 태그의 내부에 정의하며, 개수에는 제한이 없습니다.

***태그에 속성 추가 예시***`<태그명 속성명="속성값"> 콘텐츠 내용 </태그명>
또는
<태그명 속성명="속성값">`

태그명과 속성의 정의는 공백(space)로 구분하고, 큰따옴표로 속성을 표기합니다!

HTML 문법 - 주석
------------

  
> **주석이란?**  
> 사람에게는 보이지만, 컴퓨터는 읽지 못하는 코드 (메모 용도)

개발자가 코드를 작성하면서 특정 부분에 대한 메모나 필기를 하기 위해 사용하는 것을 주석이라고 합니다.

이 주석은 언어에 따라 작성법이 다른데, html에서는 아래와 같습니다.

***HTML 주석 사용 예시***`<!-- 이 태그에 작성한 내용은 모두 주석 처리 -->`
