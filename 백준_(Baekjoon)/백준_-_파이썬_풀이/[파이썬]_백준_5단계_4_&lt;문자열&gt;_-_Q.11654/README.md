**Q.11654**
-----------

알파벳 소문자, 대문자, 숫자 0-9중 하나가 주어졌을 때, 주어진 글자의 아스키 코드값을 출력하는 프로그램을 작성하시오.

**입력**

알파벳 소문자, 대문자, 숫자 0-9 중 하나가 첫째 줄에 주어진다.

**출력**

입력으로 주어진 글자의 아스키 코드 값을 출력한다.

![](https://blog.kakaocdn.net/dn/bsILrQ/btsKscCFJar/6Q16uKNUshLWkiujvkhQD0/img.png)

문자열 → 아스키코드 변환  
ord() 함수

아스키코드 → 문자열 변환  
chr() 함수

```
S = input()

print(ord(S))
```
