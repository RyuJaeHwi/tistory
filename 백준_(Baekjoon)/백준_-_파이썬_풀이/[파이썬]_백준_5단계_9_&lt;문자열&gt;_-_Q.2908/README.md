**Q.2908**
----------

상근이의 동생 상수는 수학을 정말 못한다. 상수는 숫자를 읽는데 문제가 있다. 이렇게 수학을 못하는 상수를 위해서 상근이는 수의 크기를 비교하는 문제를 내주었다. 상근이는 세 자리 수 두 개를 칠판에 써주었다. 그 다음에 크기가 큰 수를 말해보라고 했다.

상수는 수를 다른 사람과 다르게 거꾸로 읽는다. 예를 들어, 734와 893을 칠판에 적었다면, 상수는 이 수를 437과 398로 읽는다. 따라서, 상수는 두 수중 큰 수인 437을 큰 수라고 말할 것이다.

두 수가 주어졌을 때, 상수의 대답을 출력하는 프로그램을 작성하시오.

**입력**

첫째 줄에 상근이가 칠판에 적은 두 수 A와 B가 주어진다. 두 수는 같지 않은 세 자리 수이며, 0이 포함되어 있지 않다.

**출력**

첫째 줄에 상수의 대답을 출력한다.

![](https://blog.kakaocdn.net/dn/csdky4/btsKuw0zh7C/jkQHD9W6LyfvVfc06aouD0/img.png)

파이썬에서 **슬라이싱**이란, 아래와 같은 구조

여기서 'step'은 보폭 (몇개씩 끊어서 가져올지, 방향 ...)  
 + end는 슬라이싱을 끝낼 위치를 뜻함, 그러나 end에 오는 숫자 당사자는 포함X

step이 양수 = 왼 → 오 방향으로 step만큼 이동하면서 출력  
step이 음수 = 오 → 왼 방향으로 step만큼 이동하면서 출력

```
a[start : end : step]
```

위의 슬라이싱을 이용하여 숫자를 뒤집고 비교하자

```
A, B = map(int, input().split())

# 문자열을 뒤집고 다시 정수로 변환
re_A = int(str(A)[::-1])
re_B = int(str(B)[::-1])

if re_A > re_B:
    print(re_A)
else:
    print(re_B)
```
