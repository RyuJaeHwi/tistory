**Q.1157**
----------

알파벳 대소문자로 된 단어가 주어지면, 이 단어에서 가장 많이 사용된 알파벳이 무엇인지 알아내는 프로그램을 작성하시오. 단, 대문자와 소문자를 구분하지 않는다.

**입력**

첫째 줄에 알파벳 대소문자로 이루어진 단어가 주어진다. 주어지는 단어의 길이는 1,000,000을 넘지 않는다.

**출력**

첫째 줄에 이 단어에서 가장 많이 사용된 알파벳을 대문자로 출력한다. 단, 가장 많이 사용된 알파벳이 여러 개 존재하는 경우에는 ?를 출력한다.

![](https://blog.kakaocdn.net/dn/cm6TMn/btsKvInFPa5/Mj7BzKsdzAfUCmBB4y31Y1/img.png)

```
S = input()

# 비교 전 입력된 문자열을 전부 대문자로 바꿈 (대소문자 구분X)
S = S.upper()

# set() 함수를 통해 입력된 문자열에서 중복 제거 & 리스트로 변환
# why? 입력받은 문자열에 있는 각 알파벳의 개수를 세야 함
#  -> 중복되는 알파벳을 제거하고 하나씩 원본 S 문자열과 비교하면서 각 알파벳의 빈도수를 세야 함
#     (아래 for문 참조)
S_set = list(set(S))

# 알파벳 개수를 세고 새로 넣기 위한 빈 리스트 생성
newlist = []

# for문을 통해 입력된 문자열의 각 위치에 해당하는 알파벳의 개수 계산
# = 중복되는 알파벳을 제거한 리스트의 각 인덱스별 알파벳의 빈도수 계산
for i in S_set:
    S_count = S.count(i)
    newlist.append(S_count)

# newlist 리스트에서 가장 큰 값(= 가장 많이 나온 알파벳)이 2보다 크거나 같다
#  = 여러 알파벳들의 빈도수가 같다
if newlist.count(max(newlist)) >= 2:
    print("?")
else:
    # newlist에서 가장 많이 나온 알파벳(max)을 찾아서 newlist의 어느 인덱스(위치)에 있는지 확인
    #  & 해당 최빈출 알파벳을 S_set[] 리스트에서 찾아서 반환
    # ex) S_set = ['h', 'e', 'l', 'o'] / newlist = [1, 1, 2, 1]
    #     위 예시처럼 둘의 위치는 같기 때문에, index()를 이용하여 알파벳 찾기 가능
    print(S_set[newlist.index(max(newlist))])
```
